```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func main() {
	// 乱数のシード（種）を設定する
	// time.Now().UnixNano() を使うことで、プログラム実行ごとに異なるシードを与え、
	// 毎回違う乱数列が生成されるようにする
	rand.Seed(time.Now().UnixNano())

	// 0から99までの範囲のランダムな整数を生成
	// rand.Intn(n) は 0 <= x < n の範囲の整数を返す
	randomNumber := rand.Intn(100)

	fmt.Printf("生成された乱数: %d\n", randomNumber)
}
```

### 解説
- **シードの設定**:
  - `math/rand` パッケージのデフォルトの乱数生成器は、同じシード値からは常に同じ乱数列を生成します。
  - プログラムを実行するたびに異なる乱数を得るためには、`rand.Seed()` を使ってプログラムの開始時に一度だけ、一意な値（現在時刻のナノ秒などがよく使われる）をシードとして与える必要があります。
  - `init()` 関数内でシードを設定するのも一般的な方法です。

- **Go 1.20以降の注意**:
  - Go 1.20で `math/rand` パッケージが更新され、シードを設定しなくても毎回異なる乱数が生成されるようになりました。また、`rand/v2` という新しいパッケージも導入されました。
  - この解答は、それ以前のバージョンでも動作する伝統的で基本的な方法を示しています。
