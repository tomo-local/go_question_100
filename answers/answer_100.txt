```go
package main

/*
#include <stdio.h>
*/
import "C"

func main() {
	C.puts(C.CString("Hello from C!"))
}
```

### 解説
- **cgo** を使うと、Go のソースから C のコードを呼び出せます。`import "C"` の**直前**に、`/* */` で囲んだ C のコード（インクルードや関数の宣言など）を書きます。
- C の関数は `C.関数名` で呼びます。`C.puts` は C の標準ライブラリの `puts` です。Go の文字列をそのまま渡せないので、`C.CString("Hello from C!")` で C の `char*` に変換して渡します。`C.CString` は C 側でメモリを確保するため、必要に応じて C 側で解放する必要があります（`puts` は読み取り専用なのでこの例では解放していません）。
- ビルド・実行には C コンパイラ（例: gcc, clang）が必要です。環境によっては `CGO_ENABLED=1` や、インクルードパス・ライブラリの設定が必要になることがあります。cgo は Go から既存の C ライブラリを利用するときの第一歩として使われます。
