```go
package main

import "fmt"

func main() {
	// for <初期化>; <条件>; <後処理> { ... }
	for i := 0; i < 5; i++ {
		fmt.Println(i)
	}
}
```

### 解説
- Goには `for` ループしか存在せず、`while` や `do-while` はありません。しかし、`for` の文法を少し変えることで、それらのループと同じ動作を実現できます。
- この「3つのコンポーネント」形式は、C言語やJavaなど、他の多くの言語の `for` ループと同じで、最も基本的なループの形です。
  - **初期化**: ループの開始前に一度だけ実行されます。ループ内で使うカウンター変数の宣言と初期化によく使われます。
  - **条件**: ループの各反復の前に評価されます。`false` になるとループは終了します。
  - **後処理**: ループの各反復の最後に実行されます。カウンターのインクリメントなどによく使われます。
