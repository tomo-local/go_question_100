```go
package main

import "fmt"

func main() {
	counter := 0

	// for { ... }
	// 条件式を省略すると無限ループになる
	for {
		fmt.Printf("Counter is at: %d\n", counter)
		
		counter++

		if counter == 5 {
			// break文でループを強制的に抜ける
			break
		}
	}

	fmt.Println("Loop finished")
}
```

### 解説
- `for` の後に何も記述しない `for {}` という形式は、無限ループを作成します。
- 無限ループは、それ自体がプログラムをブロックするため、通常はループ内の `if` 文と `break` 文を組み合わせて、特定の条件が満たされたときにループを終了できるようにします。
- このパターンは、外部からの入力を待ち続けたり、チャネルからの受信を待ち続けたりするような、終了条件がループの外部で決定される場合に頻繁に使われます。
- `continue` 文を使うと、ループの残りの部分をスキップして次の反復に移ることができます。
