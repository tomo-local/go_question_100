```go
package main

import (
	"fmt"
	"time"
)

func main() {
	now := time.Now()

	// Goのフォーマットレイアウト "2006-01-02 15:04:05" を元に、目的の形式を指定
	layout := "2006年01月02日 15時04分05秒"
	formattedTime := now.Format(layout)

	fmt.Println(formattedTime)
}
```

### 解説
- Goの `time.Format` は、他の言語で一般的な `YYYY` や `MM` といった特別な文字を使いません。
- `2006-01-02 15:04:05` という**基準時刻**の各部分（年、月、日など）を、目的のフォーマットに合わせて並べた文字列を「レイアウト」として渡します。
- 例えば、年を `06` にすれば西暦の下2桁、`January` にすれば月の英語名フルスペルになります。
