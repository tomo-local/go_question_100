```go
package main

import (
	"fmt"
)

func main() {
	// 容量2のバッファ付きチャネル
	ch := make(chan string, 2)

	// 受信者がいなくても、バッファが空いていれば送信できる
	ch <- "first"
	ch <- "second"

	fmt.Println(<-ch)
	fmt.Println(<-ch)
}
```

### 解説
- `make(chan string, 2)` は**バッファ付き**チャネルで、最大2個まで送信をバッファに溜められます。受信者がいなくても、バッファに空きがあれば送信はブロックしません。
- 3つ目を送信すると、誰かが受信するまでブロックします。非バッファ付きチャネル（容量0）では、1つ送信するたびに受信側がいる必要があります。
- バッファ付きチャネルは、送信と受信のタイミングをある程度緩和し、一時的なバーストを吸収する用途で使われます。バッファサイズはデッドロックや遅延のトレードオフで決めます。
