```go
package main

import (
	"fmt"
	"io"
	"os"
	"strings"
)

func main() {
	r := strings.NewReader("Hello, io.Copy!")
	n, err := io.Copy(os.Stdout, r)
	if err != nil {
		fmt.Fprintf(os.Stderr, "copy error: %v\n", err)
		return
	}
	fmt.Printf("\nCopied %d bytes\n", n)
}
```

### 解説
- `io.Copy(dst io.Writer, src io.Reader) (written int64, err error)` は、`src` から `dst` へデータをコピーし、書き込んだバイト数とエラーを返します。内部でバッファを使って効率的にコピーします。
- `strings.NewReader` で作ったリーダーを `os.Stdout`（`io.Writer`）にコピーしているので、文字列がそのままコンソールに出力されます。
- コピーしたバイト数は "Hello, io.Copy!" の長さと一致します。ファイル間のコピーや HTTP のボディをファイルに保存するときなど、`io.Reader` と `io.Writer` を組み合わせる典型的なパターンです。
