```go
package main

import (
	"encoding/json"
	"fmt"
	"log"
)

// Product 構造体
// JSONに出力されるフィールドは、大文字で始まる（エクスポートされている）必要がある
type Product struct {
	ID   int    `json:"id"`
	Name string `json:"name"`
}

func main() {
	// 構造体のインスタンスを作成
	p := Product{
		ID:   1,
		Name: "Laptop",
	}

	// 構造体をJSON形式のバイトスライスに変換（マーシャリング）
	jsonData, err := json.Marshal(p)
	if err != nil {
		log.Fatalf("JSONへのマーシャリングに失敗しました: %v", err)
	}

	// バイトスライスを文字列に変換して表示
	fmt.Println(string(jsonData))
}
```

### 解説
- `json.Marshal(v interface{}) ([]byte, error)`:
  - 引数で渡されたデータ構造（通常は構造体やマップ）をJSON形式のバイトスライスに変換します。
- **構造体タグ**:
  - `json:"id"` のようにフィールドにタグを付けることで、JSON内でのキー名をカスタマイズできます。タグを付けないと、フィールド名（この場合は `ID` や `Name`）がそのままキー名になります。
- **エクスポート**:
  - 構造体のフィールドがJSONとしてエンコードされるためには、そのフィールド名が大文字で始まっている（つまり、パッケージ外にエクスポートされている）必要があります。
