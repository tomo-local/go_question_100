```go
package main

import "fmt"

func main() {
	n := 1

	// for <条件> { ... }
	// セミコロンを省略し、条件式のみを記述する
	for n <= 100 {
		n = n * 2
	}

	fmt.Println(n) // 128 が表示される
}
```

### 解説
- `for` ループの初期化ステートメントと後処理ステートメントを省略し、条件式だけを残すと、他の言語の `while` ループと等価になります。
- この形式は、ループの反復回数があらかじめ決まっておらず、特定の条件が満たされるまで処理を繰り返したい場合によく使われます。
- 内部の処理:
  - `n = 1`
  - `1 <= 100` -> `n = 2`
  - `2 <= 100` -> `n = 4`
  - `...`
  - `64 <= 100` -> `n = 128`
  - `128 <= 100` -> `false` になりループ終了
