```go
package main

import "fmt"

// 引数として `*int` 型（intへのポインタ）を取る
func double(n *int) {
	// `*n` でポインタが指す先の値にアクセスし、それを2倍にして上書きする
	*n = *n * 2
}

func main() {
	n := 5
	fmt.Printf("呼び出し前の n: %d\n", n)

	// `double` 関数に `n` のメモリアドレスを渡す
	double(&n)

	fmt.Printf("呼び出し後の n: %d\n", n)
}
```

### 解説
- Goの関数呼び出しは常に**値渡し**です。通常の `int` 型を引数に取ると、その値のコピーが関数に渡されるため、関数内で引数の値を変更しても呼び出し元の変数は影響を受けません。
- `double(n int)` のように値渡しで実装した場合:
  ```go
  func double(n int) { // nは呼び出し元の変数のコピー
      n = n * 2 // コピーを変更しているだけ
  }
  // main関数で double(n) を呼び出しても、mainのnの値は変わらない
  ```
- 関数内で呼び出し元の変数の値を直接変更したい場合は、その変数の**ポインタ**を渡す必要があります。
- これにより、`double` 関数は `n` という変数の「場所」を知ることができ、その場所にある値を直接変更することが可能になります。
