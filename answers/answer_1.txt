```go
package main

import "fmt"

func main() {
	// 1. fmt.Println の使用例
	fmt.Println("Hello", 123)

	// 2. fmt.Printf の使用例
	name := "Alice"
	age := 30
	fmt.Printf("Name: %s, Age: %d\n", name, age)
}
```

### 解説

- **`fmt.Println`**:
  - 引数をスペース区切りで出力し、最後に改行を追加します。
  - フォーマットを指定することはできず、値をそのままの形式で出力します。

- **`fmt.Printf`**:
  - 第一引数で指定されたフォーマット文字列に従って、後続の引数を埋め込んで出力します。
  - `%s` は文字列、`%d` は10進数の整数を表す「フォーマット指定子」です。
  - 自動で改行されないため、改行が必要な場合は `\n` を含める必要があります。
