問題96: タグ付き構造体

構造体のフィールドに「タグ」を付けることで、そのフィールドにメタデータを関連付ける方法を学びます。JSONのエンコード/デコードで使われる `json` タグが最も一般的です。

`User` 構造体を定義し、そのフィールドに `json` タグと `validate` タグ（架空のバリデーションライブラリ用）を付けてみましょう。
```go
type User struct {
    Name  string `json:"name" validate:"required"`
    Email string `json:"email" validate:"required,email"`
    Age   int    `json:"age,omitempty" validate:"gte=0"`
}
```
`reflect` パッケージを使って、`User` 構造体の `Name` フィールドに付けられた `json` タグの値を読み取り、コンソールに表示するプログラムを作成してください。
