問題88: テーブル駆動テスト

Goで推奨されるテストパターンの「テーブル駆動テスト」を実装します。
`add` 関数（問題86と同じ）をテストするために、テストケースを構造体のスライス（テーブル）として定義してください。

各テストケースは、入力値 `a`, `b` と期待される結果 `expected` をフィールドとして持つようにします。
`for` ループでこのテーブルを反復処理し、各テストケースについて `t.Run` を使ってサブテストを実行するプログラムを記述してください。

```go
tests := []struct {
    name    string
    a, b    int
    expected int
}{
    // ここにテストケースを定義
}
```
このような構造で実装してみましょう。
